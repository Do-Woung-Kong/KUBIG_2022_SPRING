# 국내 주식 종가 예측 스터디

<br/><br/>

## 스터디원
- 14기 김수경, 14기 김혜림
- 15기 김지호, 15기 이제윤  

<br/><br/>

## 스터디 개설 배경 및 목적

> 축적된 주식 데이터를 통계적으로 분석하여 수익률을 내는 매매 전략이 주목받고 있다.  
> 따라서 (1) 금융 도메인 전반에 대한 지식을 습득하고, (2) 실제 데이터를 이용하여 종가를 예측하고자 한다. 

<br/><br/>
